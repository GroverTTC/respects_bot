# Respects Bot
Created by: TXJ


##Installation

Required: **NodeJS 5.9+**

`npm install fs` & `npm install discordie`

Configure the `config.json` file with the fields according to your info, EX:

```json
{
	"email": "test@test.com",
	"password": "password",
	"blocklist": "./blocklist.json",
	"bot_owner": "this is your userid",
	"prefix": "??",
	"botid": "your bot user id here"
}
```
